    面版
    • 再出英雄 就是新游戏
    • 历史文物 你觉得呢？
    • 谁在创造 就是我！
    • 路过而已 给你一个后悔的机会
    • 挖脚 什么叫挖人的墙角？
    • 宝物 给你的跟班一件宝物，或者交换，或者夺走它，你的跟班又能怎样呢？
    • 召回 顾名思义吧
    • 赏赐 在游戏里，黄金可以轻易地买到一个人的心
    • 君主 富甲天下里没有办法隐藏秘密。。
    • 将军 看看都有跟班的都有谁（包括你的和人家的）？
    • 城市 地大物博啊
    • 宝物 我有什么宝物？
    • 产品 如果懂得经营之道，你可以成为一个很成功的商人。
    路过城市

       当你路过一座没有势力的城市的时候，本菜单将出现。

    • 占领 你的第一选择。
    • 买卖 从其他地方买来，在这里卖走，从这里买走，去其他地方卖
    • 粮食 没吃的了？没钱了？
    • 士兵 你的跟班的跟班，可以征召，也可以打发走。

       当你路过你的城市的时候，本菜单将出现。

    • 换防 一个跟班换另外一个跟班
    • 调度 城市太多东西了？太少东西了？

       当你路过人家的城市的时候，本菜单将出现。

    • 交税 你不识字？
    • 单挑 就是1打1，公平！
    • 战争 攻城略地！
