○测试详情
==开发信息
所有版本未使用float，未使用除SetGraphMode、SetFgColor、SetBgColor以外的非LavaX1.0的标准函数。
所有版本未使用_TEXT、_GRAPH等，符合LavaX3.0标准。
V0.36中，Sin、Cos中自变量非负，兼容TC800系列的LVM的BUG。
==关于兼容性
V0.30及以前各版本在文曲星NC3000（立天内核）实机上通过测试，
在LavaX Virtual Machine V3.1.0.050701/Windows XP SP2上通过测试及认证。
之后版本在文曲星TC800S（内核版本1.5）实机上通过测试，
在LavaX Virtual Machine V3.5.1.060615/Windows XP SP2上通过测试及认证。
==关于组件
V0.36的石英钟代码为Eastsun在易码上发表的石英钟代码的基础上修改而成，兼容TC800系列。

◎使用向导
*使用向导仅适用于当前最新一组版本。
本软件用于更改和保存、还原时间/日期。
参照更新日志，按照程序提示操作。
小时以24时制计。黑白版年份范围限制1932年至2031年。T版（灰度版）年份范围限制1901年至2050年。
设置时间/日期输入状态时按左键退出输入状态。支持按键连续响应。支持按左/右键增减数值。
按<F3>或<I>调出调出简易日历。日历年份上下限为1753年至2199年。
*时间记录文件与Zhpe的TIMETOOLS(LAVA)兼容。
