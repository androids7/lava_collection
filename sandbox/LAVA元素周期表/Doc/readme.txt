元素周期表 (详细资料版) Version 0.2

1.简介
  包含比较详细的资料的元素周期表LAVA程序

2.文件列表
  Qelem.lav	主程序文件 (下载到文曲星LAVA目录下)
  Qelem.dat	元素周期表文本资源文件 (下载到文曲星LavaData目录下)

3.使用说明
  [方向键]选择元素,程序有自己独立的键盘控制机制,可接受连续按键。
  [输入]调出主菜单。
  程序含有详细的帮助,遇到问题,只要按下[求助]键即可 :)

4.技术信息
  开发软件: GVmaker v0.9
  系统环境: Windows xp 英文版 sp2
  LAVA验证: LAVA 12K
  测试环境: NC2600c
  附：发现LAVA电脑与文曲星不兼容问题:
    1. Delay(0)在计算机和文曲星上不一样，在文曲星上会造成无法忍受的等待
    2. Crc16在计算机上正常，在文曲星上会执行非法指令导致自动关机
