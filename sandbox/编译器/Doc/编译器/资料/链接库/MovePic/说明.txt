作 者:Isword
原型:void MovePic(int x, int y, char width, char height, int data, int _data, char mode, char distance, int lasttime)
功能:移动任意位置上的任意大小的图片，而不破坏背景画面。（含掩膜作图，即透明效果）
说明:在屏幕的(x,y)坐标处绘图,图的宽为width,高为height,图形的数据地址为data，_data为掩膜轮廓的图形数据地址(只当mode=4-7时才有效）,distance需要移动的距离,lasttime移动图片一点延时lasttime个毫秒,lasttime最大取值32767。
      mode=0:图形左移
      mode=1:图形右移
      mode=2:图形上移
      mode=3:图形下移
      mode=4:掩膜图形左移
      mode=5:掩膜图形右移
      mode=6:掩膜图形上移
      mode=7:掩膜图形下移
