作 者:Eastsun
机 型:TC808
原型:void exeasm(char &buf[]);
功能:在LavaX环境下执行汇编代码。
说明:buf为汇编代码地址。汇编效率高，且可以做任何事，但无法跨平台，要合理利用。
举例:
#include <TC808.y>
char buf[] = {0x00,0x1d,0x05,0x60};//即INT $051d RTS
void main()
{
 SetScreen(0);
 printf("空间整理中...\n");
 exeasm(buf);
 printf("OK!\n");
 getchar();
}

原型:void call(int brk);
功能:在LavaX环境下调用系统中断。
说明:brk为系统中断号。
 如：0x051d为空间整理。类似汇编中的
 INT $051d
 RTS
资料:一些TC808的中断。
文件系统：
0x0501:测试超级块
0x0502:更新物理区/逻辑区映射表
0x0503:读取一页nand(物理，不做ecc校验)
0x0504:读取一页nand(物理，做ecc校验)
0x0505:读取nand(逻辑)
 $081d,$081d:nand块
 $081f,$0820偏移
 $0969,$096a:大小
 $dd,$de:开始地址
0x0506:读取一页nand(逻辑)
0x0507:读取1字节nor
0x0508:读取多字节nor
0x0509:根据指定的inode号(0x867~0x968)找出对应的目录名(0x822)
0x050a:测试目录是否可列出
0x050b:创建目录
0x050c:读取目录
0x050d:切换工作目录(0x822)
0x050e:删除文件(文件名：0x822)
0x050f:删除目录(0x822)
0x0515:打开或创建文件
 打开方式:$0860
 #$70:新建
 #$81:追加
 #$80:打开
 成功则c=0,or c=1
0x0516:读文件
 $dd,$de:开始地址
 $085d,$085e:大小
 成功则c=0,or c=1
0x0517:关闭文件
0x0518:写文件
 $dd,$de:数据地址
 $085d,$085e:大小
 成功c=0,or c=1
 读与写文件时一定要保证文件已打开
0x0519:更改文件读写指针位置
0x051a:读inode
0x051b:读一个inode
0x051c:空间整理_释放已删除空间
0x051d:空间整理_删除无用的分配表信息
0x051e:重启时空间整理A
0x051f:重启时空间整理B
0x0523:退回根目录
0x0524:格式化闪存用户区并重置文件系统（慎用格式化）
0x0525:打开文件
0x0526:读取文件
0x0527:更改文件读写指针
0x0528:资源管理器1
0x0529:快速逻辑读NAND，最多只能读255b：$081d-$081e:块；$081f-$0820:偏移
0x052a:获得及检测NAND厂商和设备号
0x052b:处理物理区/逻辑区映射表
0x052c:同上
0x052d:同上
0x052e:若指定文件>64kb，则读出其所占的块到$c000
0x052f:获取文件长度
画图部分：
0xc729:清除屏幕
0xc72a:清除文字缓冲区
0xc719:更新屏幕(类似UpdateLCD)
0xc006:获取按键值(类似getchar),按键值在a中 
0xc733:将数字转换成字符
0xc737:清除部分显存，不包括左右两边的图标
0xca02:除法
0xca03:乘法
 参数:x,a:2欲相乘数
 $80,$81:结果
0xca04:画点
0xca05:判断点
0xca0a:画非填充矩形
0xca0c:画填充矩形
0xca06:从屏幕0,0开始作矩形，不填充
0xca07:从屏幕0,0开始作填充矩形
0xca0b:画线
0xca08:从屏幕左端往屏幕上端作线
0xca0e:画圆，非填充
0xca10:画圆，填充
0xca0f:画椭圆，非填充
0xca11:画椭圆，填充
0xca09:填充作图
0xca0d:填充作图
0xca12:寄存器a=0为弹出框,a=1 or 2时为是否对话框,a=3时为选择框（此时参数有所不同）
0xca16:显示正在查找
0xca17:显示没有找到
0xcb01:系统菜单
0xcb05:输入法
 x,y:参数表
 $044a:中文输入法提示类型
 $03b3:光标类型，一般设为01
 参数表:xx (光标位置,00-3a) yy (最大可输入字符数) zz (类型，20:正常 10:只可输入数字 b0:只可输入中文 c0:只可输入英文) aa bb (帮助信息所在地址)
举例:
#include <TC808.y>
void main()
{
 SetScreen(0);
 call(0xca16);//显示正在查找
 getchar();
}
原型:void DiskReclaim();
功能:空间整理。
举例:
#include <TC808.y>
void main()
{
 SetScreen(0);
 printf("空间整理中...\n");
 DiskReclaim();
 printf("OK!\n");

}
