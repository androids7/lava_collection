水管马里奥
Lava 1.0 
50.0KB


这个游戏我做了很久，现在终于完成了（高兴）！！

我觉得这个游戏中的最难做的是马里奥的加速度（因为LavaX 1.0中不支持小数）
引擎中可以同显8个敌人，但受文区星速度限制，只放了3个（还稍慢）。


从美工到程序到测试，做完这个游戏后感觉元气大伤，去游三次泳再做新的！！！


我对自己游戏质量的要求很高，游戏中的图片都是我改了又改才出来的，游戏中物品设定也是我构思很久才做出来的。
在此声明：任何人不得盗用该游戏中的图片，仿用该游戏的设定！
